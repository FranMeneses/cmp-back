    Class OptionalJwtAuthGuard

    Guard de autenticación JWT opcional para resolvers que pueden funcionar con o sin autenticación.

    Description

    Similar a JwtAuthGuard pero no requiere autenticación obligatoria. Si hay un token válido, inyecta el usuario en el contexto. Si no hay token o es inválido, simplemente retorna null sin lanzar excepción.
