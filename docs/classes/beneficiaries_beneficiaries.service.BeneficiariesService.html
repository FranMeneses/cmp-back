    Class BeneficiariesService

    Servicio para la gestión completa de beneficiarios y contactos.

    Description

    Proporciona funcionalidades CRUD completas para:

    • Beneficiarios: entidades que reciben los beneficios de las tareas
    • Contactos: personas de referencia dentro de cada beneficiario

    Incluye mapeo automático entre formatos de base de datos y GraphQL, manejo de relaciones con tareas, y operaciones transaccionales para mantener la integridad referencial.
