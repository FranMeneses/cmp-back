    Class RolesGuard

    Guard para autorización basada en roles de usuario.

    Description

    Valida que el usuario autenticado tenga uno de los roles requeridos para acceder a un resolver o método específico. Debe usarse después de un guard de autenticación (JwtAuthGuard) para asegurar que req.user existe.
